Subway Horror

Psychological Horror · Unreal Engine 5 · Solo Project

Play on Itch.io

Overview

Subway Horror is a psychological horror game set in an abandoned subway system. You play as a lone operator navigating eerie tunnels where each stop unravels fragments of a haunting story.

Development

Created using Unreal Engine 5, this project emphasizes lighting and sound design to craft atmosphere. I built AI movement logic, environmental triggers, and custom shaders for fog and flickering lights.
