Project Details

How It Was Built

This project is a custom first-person game template built in Unreal Engine 5, designed as a reusable foundation for all my stylized PSX-inspired titles. It features a complete gameplay framework including an interaction system, inventory logic, UI menus, save/load functionality, and a modular material/shader pipeline.

The core systems are implemented using a hybrid approach: gameplay logic in C++ for performance and clarity, and Blueprints for visual, cosmetic, and UI behavior. Workflows are structured to keep logic clean and scalable — allowing the framework to be extended into future prototypes or full games without rewriting core features.

The interaction system is designed as a fully modular component, enabling dynamic objects (doors, pickups, switches, readable notes, etc.) to be implemented in no time.

The inventory system uses a data-driven architecture built around Unreal Engine data tables. This allows new items to be added or modified with minimal setup, and enables the system to scale cleanly as additional mechanics are introduced. It includes full support for storage container interactions, item consumption, equipping, and inventory transfer logic, all implemented with extensibility in mind.

A persistent save system allows full game state serialization: player stats, inventory, item states, world objects, and settings. This lets the template be used for linear or semi-open PSX-style games where continuity and progression matter.

Why

The goal of this project was to create a reusable, scalable template for my personal development pipeline — a solid starting point I can use for future indie games without rebuilding basic systems every time.

As my art style evolves, the template is continuously updated with improved master materials, character rigs, animation workflows, and PSX-style shaders. Instead of designing a single prototype, I built a flexible foundation that supports fast iteration and experimentation in any new idea I develop.

Long-term, the template will evolve into a complete toolkit for retro-styled first-person games, enabling me to create new experiences quickly while maintaining a strong and consistent visual identity.

// Initializes input mappings and binds enhanced input actions.
void APlayerCharacterController::InitializePawnInput(APawn* InPawn)
{
	EnhancedInputComponent = Cast(InputComponent);
	checkf(EnhancedInputComponent, TEXT("EnhancedInputComponent not found."));

	UEnhancedInputLocalPlayerSubsystem* InputSubsystem =
		ULocalPlayer::GetSubsystem(GetLocalPlayer());
	checkf(InputSubsystem, TEXT("Unable to get reference to EnhancedInputLocalPlayerSubsystem."));

	checkf(DefaultMappingContext, TEXT("Default Input Mapping Context missing."));
	InputSubsystem->ClearAllMappings();
	InputSubsystem->AddMappingContext(DefaultMappingContext, 0);

	// --- Bind Input Actions --- //
	if (IA_Move)
	{
		EnhancedInputComponent->BindAction(IA_Move, ETriggerEvent::Triggered, this, &APlayerCharacterController::OnMove);
	}
	if (IA_Look)
	{
		EnhancedInputComponent->BindAction(IA_Look, ETriggerEvent::Triggered, this, &APlayerCharacterController::OnLook);
	}
	if (IA_Jump)
	{
		EnhancedInputComponent->BindAction(IA_Jump, ETriggerEvent::Triggered, this, &APlayerCharacterController::OnJump);
	}
	if (IA_Crouch)
	{
		EnhancedInputComponent->BindAction(IA_Crouch, ETriggerEvent::Started, this, &APlayerCharacterController::OnCrouchStart);
		EnhancedInputComponent->BindAction(IA_Crouch, ETriggerEvent::Completed, this, &APlayerCharacterController::OnCrouchStop);
	}
	if (IA_Sprint)
	{
		EnhancedInputComponent->BindAction(IA_Sprint, ETriggerEvent::Triggered, this, &APlayerCharacterController::OnSprintStart);
		EnhancedInputComponent->BindAction(IA_Sprint, ETriggerEvent::Completed, this, &APlayerCharacterController::OnSprintStop);
	}
	if (IA_Interact)
	{
		EnhancedInputComponent->BindAction(IA_Interact, ETriggerEvent::Started, this, &APlayerCharacterController::OnInteractStart);
		EnhancedInputComponent->BindAction(IA_Interact, ETriggerEvent::Completed, this, &APlayerCharacterController::OnInteractStop);
	}
	if (IA_ToggleInventory)
	{
		EnhancedInputComponent->BindAction(IA_ToggleInventory, ETriggerEvent::Started, this, &APlayerCharacterController::OnToggleInventory);
	}
}

// Removes item from inventory when storing (e.g., into a chest or container).
void APlayerCharacter::StoreItem(UItemBase* ItemToStore, const int32 QuantityToStore)
{
	if(InventoryComponent->FindMatchingItem(ItemToStore))
	{
		InventoryComponent->RemoveAmountOfItem(ItemToStore, QuantityToStore);
	}
	else
	{
		UE_LOG(LogTemp, Warning, TEXT("Item to drop was null"));
	}
}
