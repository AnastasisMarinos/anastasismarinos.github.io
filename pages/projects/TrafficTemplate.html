Project Details

How It Was Built

This project is a modular traffic and vehicle system prototype built in Unreal Engine 5, designed for open-world environments and sandbox projects. All core logic, vehicle physics, traffic behavior, and collision response is implemented using and split up between visual logic (Blueprints) and core systems (C++/Mass AI).

Vehicle movement uses simplified physics implemented through Unreal’s Chaos system, with acceleration, braking, steering curves, and suspension. Each vehicle is driven by a shared component set that handles wheel rotation, throttle input, speed calculation, and basic damage responses.

The traffic system is powered by the Mass AI plugin. Vehicles follow splines or predefined navigation lanes, obey speed limits, slow down when approaching other vehicles, and react to obstacles. Traffic agents are spawned, pooled, and despawned dynamically to support large open-world maps without performance spikes. Collision events feed into a simple crash-reaction system that applies impulses and temporary handling penalties.

All systems are built to be modular. New vehicles can be added by assigning a data asset defining physics settings and a behavior profile. New traffic patterns can be generated by rearranging spline networks and Mass entities.

Why

The goal of this project was to create a lightweight, reusable traffic and vehicle framework that can be dropped into any open-world Unreal Engine project without relying on heavy third-party systems. I wanted to understand the fundamentals of vehicle physics, Mass AI workflows, agent spawning, and performance-friendly simulation of large numbers of actors.

Long-term, this system is intended to become a polished, standalone template for indie developers. The plan is to replace any placeholder or copyrighted vehicle assets, expand physics interactions, polish the AI rules, and release a clean version that can serve as a starting point for open-world prototypes.

Vehicle Base

// Slow down vehicle rotation while in air by increasing the meshes angular dampening. 
void AVehicleBase::DampenWhileInAir()
{
	static bool bWasOnGround = true;
	bool bOnGround = MovementComponent->IsMovingOnGround();

	if (bOnGround != bWasOnGround)
	{
		GetMesh()->SetAngularDamping(bOnGround ? 0.f : 3.f);
		bWasOnGround = bOnGround;
	}
}

Player Vehicle Base

// Realign Camera Back To Its Original Position While Driving.
void APlayerVehicleBase::RealignCamera(float DeltaSeconds)
{
	// Get vehicle forward speed (in cm/s)
	float ForwardSpeed = MovementComponent->GetForwardSpeed();
	
	if (ForwardSpeed != 0.0f)
	{
		// Ignore small jitter
		if (FMath::Abs(ForwardSpeed) < 5.0f)
			return;

		// Target yaw depending on direction
		float TargetYaw = (ForwardSpeed < -10.0f) ? 180.0f : 0.0f;

		FRotator CameraRotation = CameraArm->GetRelativeRotation();

		// Use FixedTurn to smoothly rotate shortest path without 360° flip
		float NewYaw = FMath::FixedTurn(CameraRotation.Yaw, TargetYaw, CameraRealignSpeed * DeltaSeconds * 50.0f);

		CameraArm->SetRelativeRotation(FRotator(0.0f, NewYaw, 0.0f));
	}
}
