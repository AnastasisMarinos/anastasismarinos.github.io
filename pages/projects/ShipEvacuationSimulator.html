Project Details

How It Was Built

Developed in UE5 using EOS for peer-to-peer multiplayer. This project features a full asymmetrical horror gameplay loop.

This project is an asymmetrical horror prototype built in Unreal Engine 5 using a clean separation between visual logic (Blueprints/UI) and core gameplay systems (C++). The core loop is modeled after Dead by Daylight: survivors repair generators, power the exit, heal each other, and avoid being caught and hooked by the killer.

Most gameplay rules — interaction flow, survivor states (healthy/injured/crawling/carried/hooked), health progression, generator repair and sabotage, hook and wiggle logic, etc. — are implemented and replicated in C++ components. Those components drive authoritative game state on the server, and then push cosmetic/UI updates to clients. That includes survivor health bars, skill checks, action progress bars, and endgame triggers (survivors escape vs. killer wins).

The match flow runs on Epic Online Services (EOS). The server creates or joins EOS sessions, assigns one player as the killer and the rest as survivors, and replicates win conditions. All survivors are dead vs. the exit powered and survivors escaped.

Why

The goal of this project was to learn by recreating the full loop of an online asymmetrical horror game: networking, authority, session management, replicated combat, and interaction systems. I wanted to build something more complex than a “simple demo” and closer to an actual multiplayer game that I can experience with my friends.

Long-term, the goal is to turn this into a reusable asymmetric horror template. The plan is to replace any copyrighted placeholder content (like the Silent Hill: Midwich Elementary map and Pyramid Head killer) and ship a clean, original base that other devs can build on.

Built With

• Unreal Engine 5 (C++ gameplay framework + Blueprint-driven UI)
• Epic Online Services (sessions, lobbies, online identity, player presence)
• Niagara for hit / blood particle effects and generator VFX
• Custom movement components for both Survivor and Killer (sprint, crawl, stun states)
• Replicated animation montage system for melee, carry, hook, etc.
• Blender for modeling and kitbashing level content
• Retro PSX-style art direction (low poly, crunchy textures, heavy atmosphere)
• FL Studio for sound effects and audio work

void ASurvivorCharacter::BeginPlay()
{
	Super::BeginPlay();

	if (HasAuthority())
	{
		if (APlayerGameState* GS = GetWorld()->GetGameState())
		{
			GS->RegisterSurvivor(this);
		}
	}

	if (HealthComponent)
	{
		HealthComponent->OnHealthStateChanged.AddDynamic(this, &ASurvivorCharacter::OnHealthStateChanged);
	}

	if (SkillCheckComponent)
	{
		SkillCheckComponent->OnStartNotify.AddDynamic(this, &ASurvivorCharacter::NotifySkillCheck);
		SkillCheckComponent->OnFail.AddDynamic(this, &ASurvivorCharacter::FailSkillCheck);
		SkillCheckComponent->OnSuccess.AddDynamic(this, &ASurvivorCharacter::SucceedSkillCheck);
		SkillCheckComponent->OnGreat.AddDynamic(this, &ASurvivorCharacter::GreatSkillCheck);
		SkillCheckComponent->OnStartSkillCheck.AddDynamic(this, &ASurvivorCharacter::StartSkillCheck);
		SkillCheckComponent->OnUpdateFailTimer.AddDynamic(this, &ASurvivorCharacter::UpdateFailTime);
	}
}

void ASurvivorCharacter::GetLifetimeReplicatedProps(TArray& OutLifetimeProps) const
{
	Super::GetLifetimeReplicatedProps(OutLifetimeProps);

	DOREPLIFETIME(ASurvivorCharacter, bIsRepairing);
	DOREPLIFETIME(ASurvivorCharacter, bIsPoweringGates);
	DOREPLIFETIME(ASurvivorCharacter, bIsBeingHealed);
	DOREPLIFETIME(ASurvivorCharacter, bIsHealingOther);
	DOREPLIFETIME(ASurvivorCharacter, PlayMontageInfo);
}

// React to health state changes.
void ASurvivorCharacter::OnHealthStateChanged(EHealthState NewState)
{
	switch (NewState)
	{
	case EHealthState::Carried:
		{
			// When picked up by the killer: force out of crawl, disable collisions and movement,
			// and attach to the killer’s mesh (carry socket) once we have the reference.
			if (SurvivorMovement) { SurvivorMovement->SetCrawl(false); }

			GetCapsuleComponent()->SetCollisionEnabled(ECollisionEnabled::NoCollision);
			if (UCharacterMovementComponent* CM = GetCharacterMovement())
			{
				CM->StopMovementImmediately();
				CM->SetMovementMode(MOVE_None);
				CM->Deactivate();
			}

			if (AKillerCharacter* Killer = HealthComponent ? HealthComponent->GetActiveKiller() : nullptr)
			{
				if (USkeletalMeshComponent* KMesh = Killer->GetMesh())
				{
					AttachToComponent(KMesh,
						FAttachmentTransformRules::SnapToTargetNotIncludingScale,
						TEXT("carry_socket"));
				}
			}
			break;
		}
	case EHealthState::Hooked:
	{
		// While hooked, attach to the hook mesh (socket), disable movement & collision,
		// and make sure we are not in crawl mode anymore.
		AHook* CurrentHook = HealthComponent ? HealthComponent->GetActiveHook() : nullptr;
		if (CurrentHook && CurrentHook->HookMesh)
		{
			DetachFromActor(FDetachmentTransformRules::KeepWorldTransform);
			AttachToComponent(CurrentHook->HookMesh, FAttachmentTransformRules::SnapToTargetNotIncludingScale, TEXT("hook_socket"));
			GetCapsuleComponent()->SetCollisionEnabled(ECollisionEnabled::NoCollision);
			GetCharacterMovement()->SetMovementMode(MOVE_None);
			GetCharacterMovement()->Deactivate();
		}
		if (SurvivorMovement) { SurvivorMovement->SetCrawl(false); }
		break;
	}
	case EHealthState::Crawling:
	{
		// On the ground: enable crawl locomotion, re-enable movement/collision, and detach from anything.
		if (SurvivorMovement) { SurvivorMovement->SetCrawl(true); }
		DetachFromActor(FDetachmentTransformRules::KeepWorldTransform);
		GetCapsuleComponent()->SetCollisionEnabled(ECollisionEnabled::QueryAndPhysics);
		GetCharacterMovement()->SetMovementMode(MOVE_Walking);
		GetCharacterMovement()->Activate();
		break;
	}
	default:
	{
		// Healthy/Injured baseline: ensure normal walking and collisions, detach from sockets,
		// and force a network update to quickly inform other clients of our new mode.
		DetachFromActor(FDetachmentTransformRules::KeepWorldTransform);
		SetReplicateMovement(true);
		ForceNetUpdate();
		GetCharacterMovement()->StopMovementImmediately();
		GetCharacterMovement()->ClearAccumulatedForces();
		GetCharacterMovement()->SetMovementMode(MOVE_Walking);
		GetCharacterMovement()->Activate();
		GetCapsuleComponent()->SetCollisionEnabled(ECollisionEnabled::QueryAndPhysics);
		if (SurvivorMovement) { SurvivorMovement->SetCrawl(false); }
		break;
	}
	}
}
